    • Range - The set of possible outputs of a function
    • Root - The values of x for which the output is 0 (\(f(x) = 0\))

The Linear Function

  • The Linear Function

    • Concept
      • The univariate linear function describes the output as a linear function of a variable (variable x)
    • Mathematics
      • \(f(x) = ax + b\)

  • The Linear Equation

    • Concept
      • The univariate linear equation is an equation that describes a variable (variable y) as a linear function of another variable (variable x)
    • Mathematics
      • \(y = ax + b\)
        • Where
          • \(y\)
            • Variable y (A variable represented on the y-axis)
            • This variable is usually the output, which is also called the outcome variable or the dependent variable (DV) in mathematical modelling
          • \(x\)
            • Variable x (A variable represented on the x-axis)
            • This variable is usually the input, which is also called the predictor variable or the independent variable (IV) in mathematical modelling
          • \(a\)
            • The coefficient
          • \(b\)
            • The intercept

    • The elements and the line

    • Characteristics of the line
      • The slope
        • Concept
          • The slope of the line describes the nature of the relationship between the x and the y variable

  • The coefficient defines the slope of the line
    - Defines the unit by which the line is shifted vertically (along the y-axis) from the origin - The y coordinate of the point of the line when x = 0

  • The a coefficient determines the slope of the line (the slope of the line represents the nature of the relationship )

  • The sign of the a coefficient determines the direction of the slope

  • The sign of the a coefficient determines the direction of the slope - Positive sign (\(+\))/slope - The x and y variable are positively related to each other

    • A slope with a negative sign (or a negative slope) - The x and y variables are negatively related to each other

  • The magnitude of the a coefficient

  • The slope of the line describes the magnitude of the relationship between the x and y variable - That is, how strongly x and y varies together
